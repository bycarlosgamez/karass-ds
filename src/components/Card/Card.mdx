## ðŸŽ¤ Description

Cards are versatile UI elements designed to present media, text, and interactive content in a structured way. This Card pattern includes several modular subcomponents:

- **CardImage**: Displays a media thumbnail or banner with optional link wrapping.
- **CardHeader**: Text heading styled for prominence.
- **CardBody**: Description or primary content area.
- **CardCTA**: Link or button-style call-to-action with optional SVG icon.
- **CardContent**: Layout wrapper for internal sections.

Each subcomponent is customizable, allowing developers to craft blog previews, dashboard widgets, or product teasers with ease.
