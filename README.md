# React + TypeScript + Vite + Storybook

A component library that uses React, Typescript, Vite and Storybook.

### Get started

`npm install`

### Running Storybook

`npm run storybook`

### Running Tests

`npm run test:ui`

### Building Storybook

`npm run build-storybook`

### Build Component Library

`npm run build`
